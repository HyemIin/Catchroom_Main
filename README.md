## ⚙️ 프로젝트 세팅

- 자바 버전 : **17**
- 서버 : AWS EC2
- 스프링 버전 : **Spring Boot 3**
- 데이터베이스 : Mysql, Redis , MongoDB 
- 문서화 도구 : Swagger
- 채팅 : Web Socket,STOMP,Redis Pub/Sub
- 의존성
    - Data-JPA
    - Data-Redis
    - QueryDSL
    - Validation
    - Security
    - Lombok
    - jwt
    - Spring Cloud
